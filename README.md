## Usage
Open Up terminal and type ->
```python
git clone https://github.com/BLUE-DEVIL1134/ProX-WebApp.git
cd ProX-WebApp
cd nodeServer
node index.js
```

Now Open Up another terminal (cmd) and enter ->
```python
start index.html
```
